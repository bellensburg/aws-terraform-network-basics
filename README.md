# aws-terraform-network-basics

# AWS Terraform Network Examples (Booking.com Style)

This repository demonstrates AWS network infrastructure using Terraform, 
focused on real-world cloud networking patterns:

- Multi-AZ VPC
- Public + private subnets
- Transit Gateway attachments
- Route table propagation
- Gateway Load Balancer endpoints (for cloud firewalls)
- Internet/NAT routing logic

The goal is to show:
- Understanding of AWS network architecture
- Ability to structure Terraform modules
- Cloud security patterns (GWLB)
- Transit Gateway connectivity
- IaC mindset for large-scale environments
